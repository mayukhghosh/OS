The program can be run by executing the following command:
javac *.java (to compile all the necessary classes)
java Main [--verbose] input-[Input number].txt

For example:
java Main --verbose input-1.txt

Note: It is assumed that the input files do not have any paranthesis in them unlike in the ouptput files provided.

Also, the processes in the final summary are printed in the sorted order according to their arrival.

It also assumes that the random-numbers.txt and all other input files are in the same directory as the one it is being executed from.
